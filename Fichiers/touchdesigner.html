Weather API with kinect sensor

This is an experimentation using an API to retreive datas online. The goal was to get realtime input of meteo data. Every 10 mins a request is sent and the temperature is update on the screen. 3 cities are used in this exemple but it can be much more and any cities in the world.

The kinect is use as as sensor who detect presence and position of the user. The video shows the kinect input as I was not able to film it in real condition. The position affect the data as some of it only appear when the user face it.
